ГородСтрой Pro — статический сайт
==================================

Состав:
  index.html      — разметка страницы
  style.css       — стили
  script.js       — логика (меню, модалки, калькулятор, FAQ)
  assets/         — фотографии (hero, p1–p3 для портфолио)

Как запустить:
  1. Просто откройте index.html в браузере.
  2. Для размещения на хостинге загрузите все файлы в корень
     (или любую папку) — никаких зависимостей и сборки не нужно.

Шрифты Inter и Manrope подгружаются с Google Fonts через CDN.
Если нужна работа без интернета — закомментируйте <link> на fonts.googleapis.com
в index.html, браузер использует системный sans-serif.
