ГородСтрой Про — статический сайт (HTML + CSS + JS)

Состав:
  index.html       — разметка
  style.css        — стили
  script.js        — интерактив (модалки, FAQ, формы, скролл, моб-меню)
  images/          — фотографии (13 шт.)

Как запустить локально:
  Откройте index.html двойным кликом в браузере.

Как разместить на хостинге:
  Залейте все файлы (включая папку images) в корень сайта по FTP/SFTP
  или через панель хостинга. Подходит любой статический хостинг:
  Beget, reg.ru, Timeweb, GitHub Pages, Netlify, Cloudflare Pages.

Формы заявок: сейчас работают как демо (показывают сообщение «Спасибо»).
Чтобы заявки приходили на почту/Telegram — подключите свой обработчик
(например, Formspree, FormBold, formsubmit.co) в атрибут action у форм.
